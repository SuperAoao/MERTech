# Gitmoji commit prefixes (copy/paste)

Use these at the start of your commit message, e.g.:

`✨ Add CBF player-based 5-fold training script`

## Most common

- `✨` **feat**: new feature / capability
- `🐛` **fix**: bug fix
- `📝` **docs**: documentation only
- `♻️` **refactor**: refactor without changing behavior
- `🚀` **perf**: performance improvement
- `✅` **test**: add/update tests
- `🔧` **chore**: tooling, configs, small maintenance
- `⬆️` **deps**: upgrade dependencies
- `⬇️` **deps**: downgrade dependencies
- `🔒` **security**: security fix

## Code quality / maintenance

- `🎨` **style**: formatting, whitespace, lint (no logic change)
- `🚨` **lint**: fix linter warnings/errors
- `🔥` **remove**: remove code/files
- `💩` **debt**: intentionally bad code / quick hack (avoid if possible)
- `💡` **comments**: add/update comments

## Build / CI / release

- `🏗️` **build**: build system changes
- `👷` **ci**: CI changes
- `📦` **package**: add/update package artifacts
- `🔖` **release**: version tag / release

## Data / assets / model artifacts

- `📦` **assets**: add/update assets (images, audio samples, etc.)
- `🧠` **model**: model weights / checkpoints / training artifacts
- `📊` **metrics**: evaluation scripts/metrics changes
- `🗃️` **data**: dataset or data pipeline changes

## UX / UI (if applicable)

- `💄` **ui**: UI/UX tweaks
- `♿️` **a11y**: accessibility improvements

## Dev experience

- `🔍` **debug**: add/remove debug logs
- `🚧` **wip**: work in progress
- `📈` **analytics**: add/update analytics / tracking

## Git operations

- `🔀` **merge**: merge branches
- `⏪️` **revert**: revert changes
- `🙈` **gitignore**: update `.gitignore`

## Infra / deploy

- `🐳` **docker**: Docker related
- `☸️` **k8s**: Kubernetes related
- `🚚` **move**: move/rename files
- `🚑️` **hotfix**: critical hotfix

---

Tip: keep titles short and action-oriented: “Add…”, “Fix…”, “Refactor…”.
